## Verb - RUN: Wagatano - Friday

| Kikuyu Word/Phrase | English Translation | Notes |
| :--- | :--- | :--- |
| **Teng'era** | run, **Gũtengera** - to run | Base verb and its infinitive form. |
| **Teng'erithia** | cause to run | Causative form of the verb. |
| **Teng'eria** | run after another (chase) | Applicative form, indicating action directed towards someone/something. |
| **Teng'erania** | Chase others about | Reciprocal or associative form. |
| **Kamau nĩ mũtengeri** | Kamau is a runner, athlete | Implies a fast runner if the root word "run" is stressed. |
