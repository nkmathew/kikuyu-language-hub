## Kĩkũyũ Vocabulary: Largeness and Smallness

The examples show how prefixes and adjectives change based on the noun class to denote size.

| Kĩkũyũ Term | English Translation | Noun Class/Size |
| :--- | :--- | :--- |
| **Mũtĩ mũceke** | A thin/small tree | Thin/Small (Mũ- class) |
| **Mũtĩ mwarĩĩ** | A large/tall tree | Large/Tall (Mũ- class) |
| **Gĩkabu gĩceke** | A thin/small basket | Small/Thin (Gĩ- class) |
| **Gĩkabu kĩarĩĩ** | A large/tall basket | Large/Tall (Gĩ- class) |
| **Kĩmũbira** | A very large ball | Very Large (Kĩ- class augmentative) |
| **Mũbira** | A ball | Normal size (Mũ- class) |
| **Kamũbira** | A small ball | Small (Ka- class diminutive) |
| **Ng'ombe noru** | A fat cow | Fat (N- class adjective) |
| **Ng'ombe hũnju** | A thin/skinny cow | Thin/Skinny (N- class adjective) |
| **Mũndũ mũkũhĩ** | A short person | Short (Mũ- class adjective) |
| **Mũndũ mũraihu** | A tall person | Tall (Mũ- class adjective) |
| **Mũndũ mũceke** | A thin/slim person | Thin/Slim (Mũ- class adjective) |
| **Mũndũ mũtungu** | A fat person | Fat (Mũ- class adjective) |
