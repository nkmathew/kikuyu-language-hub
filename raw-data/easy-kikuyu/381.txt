## Juma - Saturday: Mawĩra - Jobs

| Kikuyu Sentence/Phrase | English Translation | Notes |
| :--- | :--- | :--- |
| **(Nĩĩ) Ndĩ mwendia** | I am a seller | |
| **(Nĩĩ) Ndĩ Mũgũrĩ** | I am a buyer | |
| **Ngũraga ngĩendagia** | I buy and sell | Verb conjugation for "I buy and I sell." |
| **(Wẽẽ) wĩ Mũtongoria** | You are a leader | |
| **(Wẽ) nĩ Mũcori** | he/she is an artist | **Mũcori** is a general term for one who draws or paints. |
| **(Wẽ) nĩ Mũcori wa nyũmba** | he/she is an architect | Literally, a "drawer of houses." |
| **(Inyuĩ) mwĩ atongoria** | You (plural) are leaders | |
| **Mũthigari** | police (or any uniformed security person) | |
| **Ũyũ nĩ mũthigari wa Mbĩrarũ** | this is a soldier | **Mbĩrarũ** means military/army. |

***

The optional subject pronouns (in brackets) can be ignored without changing the meaning of the sentences.
