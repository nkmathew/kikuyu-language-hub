## Nuclear Family and Dialogue

This vocabulary and dialogue are taken from the image illustrating the family:

| Kikuyu Phrase | English Translation | Context/Notes |
|:---|:---|:---|
| **Ũyũ nĩ baba** | This is father | Father is taking photographs. |
| **Ũyũ ũngĩ nĩ maitũ** | This other one is mother | |
| **Tũrũgamĩte na mwarĩ wa maitũ** | We are standing with my sister | Literally: "We are standing with the daughter of my mother." |
| **Ũcio mũraihu nĩ mama** | The tall one is my uncle (mother's side) | **Mama** refers to the maternal uncle. |
| **Baba arahũra mbica** | Father is taking photographs | |
| **Mwarimũ:** Maina rũgama | **Teacher:** Maina, stand up | |
| **Maina:** Nĩ ndarũgama mwarimũ | **Maina:** I have stood up teacher | |
| **Mwarimũ:** Ũyũ nũũ ? | **Teacher:** Who is this | Asking about the father. |
| **Maina:** Ũyũ nĩ baba | **Maina:** This is father | |
| **Mwarimũ:** Ũyũ ũngĩ nĩ nũũ ? | **Teacher:** Who is this other one? | Asking about the mother. |
| **Maina:** Ũyũ ũngĩ nĩ maitũ | **Maina:** This other one is mother | |
| **Mwarimũ:** Mũrũgamĩte naũ? | **Teacher:** Who are you standing with? | |
| **Maina:** Tũrũgamĩte na Mwarĩ wa maitũ | **Maina:** We are standing with my sister | |
| **Mwarimũ:** Ĩkara thĩ | **Teacher:** Sit down | |
