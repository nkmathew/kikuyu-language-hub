# BASIC LANGUAGE INFORMATION
# =============================================================================

# Language Name: Gĩkũyũ (also known as Kikuyu, Gĩgĩkũyũ)
# Pronunciation: [ɣēkōjó]
# Native to: Kenya, Central Province
# Ethnicity: Agĩkũyũ (Kikuyu people)
# Native speakers: 6.6 million (2009 census)

# Language Family: Niger-Congo > Atlantic-Congo > Volta-Congo > Benue-Congo > 
#                  Bantoid > Southern Bantoid > Bantu > Northeast Bantu > 
#                  Upland Bantu > Thagiicu > Kikuyu-Temi > Kikuyu

# ISO Codes: ISO 639-1: ki, ISO 639-2: kik, ISO 639-3: kik
# Glottolog: kiku1240, Guthrie Code: E.51

# =============================================================================
# DIALECTS
# =============================================================================

# Gichugu
# Mathira  
# Ndia
# Northern Gikuyu
# Southern Gikuyu

# =============================================================================
# WRITING SYSTEMS
# =============================================================================

# Latin script (primary)
# Gicandi script (traditional)

# Latin Alphabet: a b c d e g h i ĩ j k m n o r t u ũ w y
# Note: Does not use f l p q s v x z; adds ĩ and ũ

# =============================================================================
# PHONOLOGY - VOWELS
# =============================================================================

# Vowel Chart:
# High: i (front), u (back)
# Mid-high: e ⟨ĩ⟩ (front), o ⟨ũ⟩ (back)  
# Mid-low: ɛ ⟨e⟩ (front), ɔ ⟨o⟩ (back)
# Low: a (central)

# =============================================================================
# PHONOLOGY - CONSONANTS
# =============================================================================

# Nasals:
# m (bilabial), n (dental/alveolar), ɲ ⟨ny⟩ (palatal), ŋ ⟨ng'⟩ (velar)

# Plosives:
# t (voiceless dental/alveolar), k (voiceless velar)
# ᵐb ⟨mb⟩ (voiced prenasalised bilabial)
# ⁿd ⟨nd⟩ (voiced prenasalised dental) 
# ᵑɡ ⟨ng⟩ (voiced prenasalised velar)

# Affricates:
# ᶮdʒ ⟨nj⟩ (palatal)

# Fricatives:
# ʃ ⟨c⟩ (voiceless palatal), h (voiceless glottal)
# β ⟨b⟩ (voiced bilabial), ð ⟨th⟩ (voiced dental), ɣ ⟨g⟩ (voiced velar)

# Liquids:
# ɾ ⟨r⟩ (dental/alveolar)

# Approximants:
# j ⟨y⟩ (palatal), w (velar)

# Note: Prenasalised consonants (/ᵐb ⁿd ᶮdʒ ᵑɡ/) are often pronounced as [b d dʒ ɡ]

# =============================================================================
# TONES
# =============================================================================

# Two level tones: high and low
# Low-high rising tone
# Downstep

# =============================================================================
# BASIC GRAMMAR
# =============================================================================

# Word Order: Subject-Verb-Object (SVO)
# Uses prepositions (not postpositions)
# Noun phrase order: Noun + Possessive + Demonstrative + Adjectives + Quantifiers + Numerals

# =============================================================================
# NOUN CLASSES (17 Classes)
# =============================================================================

# Class 1: mũ- (animate/human, singular)
# Class 2: a- (animate/human, plural)
# Class 3: mũ- (nature/landscape, singular)
# Class 4: mĩ- (nature/landscape, plural)
# Class 5: rĩ-/i- (plant/landscape, singular)
# Class 6: ma- (plant/landscape, plural)
# Class 7: gĩ-/kĩ- (augmentative, singular)
# Class 8: ci-/i- (augmentative, plural)
# Class 9: ∅ (animals, loanwords, body parts, singular)
# Class 10: ∅ (animals, loanwords, body parts, plural)
# Class 11: rũ- (long, thin, string-like, singular)
# Class 12: ga-/ka- (diminutive, singular)
# Class 13: tũ- (diminutive, plural)
# Class 14: ∅ (abstract concepts)
# Class 15: gũ-/kũ- (body parts, verbal infinitives)
# Class 16: ha- (definite locative)
# Class 17: kũ-/gũ- (indefinite locative)

# BASIC VOCABULARY - COMMON PHRASES

# Greetings and Social Interactions:
# How are you? -> Ũhoro waku / kũhana atĩa?
# How are you doing? -> Ũrĩ mwega? / Wĩ mwega
# I am good -> Ndĩ mwega
# Are you a friend? -> Wĩ mũrata?
# Thank you -> Thengiũ / Nĩ wega / Nĩ ngaatho
# I give thanks -> Nĩndacokia ngatho
# I'm blessed -> Ndĩĩ mũrathime
# Bye, be blessed -> Tigwo na wega / Tigwo na thaayũ
# Go in peace -> Thiĩ na thaayũ

# Expressions of Need and Emotion:
# Give me water -> He maaĩ
# I am hungry -> Ndĩ mũhũtu
# Help me -> Ndeithia
# I love you -> Nĩngwendete
# Give me money -> He mbeca / He mbia

# Commands and Requests:
# Come here -> Ũka haha
# Stop nonsense -> Tiga wana / tiga ũrimũ
# Don't laugh -> Ndũgatheke

# Communication:
# I will phone you -> Nĩngũkũhũrĩra thimũ
# You are learned -> Wĩ mũthomu

# NUMBERS AND COUNTING
# =============================================================================

# Numerals 11-19: formed with construction "ten and X"
# Final numeral agrees with noun being counted
# If final numeral is 1: agrees with singular class (even in compound numbers)

# =============================================================================
# VERB MORPHOLOGY
# =============================================================================

# Verb markings include:
# - Focus
# - Noun class agreement
# - Negation
# - Reflexivity
# - Reciprocality
# - Causativity
# - Intensive meanings
# - Reversive meanings
# - Applicative (valency increasing) meanings

# Tenses:
# - Past, Present, Future
# - Remote, Near, Current

# Aspects:
# - Habitual/Imperfective
# - Completive
# - Perfect
# - Progressive (unmarked)
# - Sequential (subtype of progressive)
# - Persistive (continuous until time of speaking)

# Voices:
# - Active
# - Passive
# - Middle (intermediate connotation)

# Subject Agreement:
# - Special particles for 1st and 2nd person
# - Noun class-based agreement for other persons
# - Can agree with all arguments, but less common with non-human nouns
